會(ē)做官(tsò-kuann)，會(ē)察理(tshat-lí)。

會(ē)行(kiânn) 行晬一(tsè-it)，袂(bē)行(kiânn) 行晬七(tsè-tshit)。

會(ē)扛轎(kng-kiō)，才(tsiah)通(thang)開(khui)轎間(kiō-king)。

會(ē)，成丁(sîng-ting)；袂(bē)，放銃(pàng-tshìng)。
（即句嘛有講做：「會，成丁；袂，房清。」跋筊儂(pua̍h-kiáu-lâng)一注(tsi̍t-tù)盡拚。）

會(ē)算(sǹg)袂(bē)除(tî)，糶米(thiò-bí)換蕃薯。

會食酒(tsia̍h-tsiú)，免濟(tsē)菜。

會遮(jia)著(tio̍h)頭，袂遮(jia)著(tio̍h)尾。

會曉(ē-hiáu)偷食(thau-tsia̍h)，袂曉(bē-hiáu)拭嘴(tshit-tshuì)。

會曉(ē-hiáu)洗面(sé-bīn)，免偌儕(juā-tsē)水。

會得(ē-tit)開刀(khui-to)，袂得(bē-tit)入鞘(ji̍p-siò)。

2023年5月註：

十幾年前用當時个HTML <ruby> tag 舊規格，後來个瀏覽器就無法度正常顯示矣。 趁即擺徙過來台語文理想國新站， 共陳建中先生請教了後，改用新版个<ruby>規格，就會得顯示標音效果矣。 不而過，預設值毋是真理想，連紲兩个台羅音節，雖然分別屬佇無仝詞， 煞真容易連做伙、袂閬縫。陳先生教我講會使佇 CSS 設定 ruby 元素个 margin值，我先設1px。 

ruby {
    margin: 1px;
}

網頁規格HTML當中所講个ruby就是指親像華文漢字加注音符號个時， 注音符號出現佇漢字邊=仔个效果。欲扑ruby標音个HTML檔，用手工傷忝頭， 愛利用家私頭仔，像「信望愛台語客語輸入法」个功能設定內底就有提供 「開啟 ruby 模式」予咱勾選，閣點選「使用 HTML ruby 模版」， 了後咱咧扑台語漢字个時，輸入法就會自動替咱加 ruby tag个羅馬字（台羅或是教羅）注音矣。若平常時欲使用信望愛輸入法， 愛會記得將「開啟 ruby 模式」个勾仔取消。

假使咱佇一篇文章內，干但一部份个台語漢字頂頭需要加注ruby台羅音， 按呢攏用信望愛輸入法就會真費氣，因為著愛定定入去設定內底將「開啟 ruby 模式」 連鞭開連鞭關（關起來才會使扑單純个漢字抑是羅馬字）。 因為我扑台語漢字是用嘸蝦米輸入法為主，所以是佇嘸蝦米佮信望愛兩種輸入法 中間切換，按呢顛倒比一直入去信望愛个設定內底開關[khui-kuainn]較快活。
