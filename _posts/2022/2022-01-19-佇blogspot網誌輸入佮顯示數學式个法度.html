研究規工，總算試出來矣！佇 blogspot.com 網誌輸入數學式个法度，著利用 MathJax 配合 Mathcha：

參考 Is there any mathematical blog support latex? 先點 blogspot 右上角「設計」，閣點「主題 -> 自訂(編輯HTML)」，佇 <head>下跤增加 MathJax 所提供个數學式轉換服務佮設定：

<script type="text/x-mathjax-config">
  MathJax.Hub.Config({
  extensions: ["tex2jax.js"],
   jax: ["input/TeX", "output/HTML-CSS"],
   tex2jax: {
     inlineMath: [[ '\$','\$']],
     displayMath:  [['\$\$','\$\$']],
     processEscapes: true
   },
   "HTML-CSS": { availableFonts: ["TeX"] }
  });
</script> 

<script type="text/javascript"          
 src="https://cdn.jsdelivr.net/npm/mathjax@2/MathJax.js">
</script>

存起來。

了後去 Mathcha - Online Mathematics Editor 利用伊線頂个數學式編輯器，將咱个數學文章佇遐扑予好勢，規篇反白，選 Menu -> Export Selection to Latex -> Copy to Clipboard。了後來 blogspot.com 即爿，開一篇新文章，佇「HTML檢視」模式，將電腦 Clipboard（剪貼簿）內底个文字貼過來，適當補上一寡 <p> tags 來將文章分段予好看。最後就會使得發布矣。

注意：按呢設好了後，因為伊是用前後个 \$ 記號去表示其中所夾--个是需要特別處理轉換个數學式，所以此去若欲扑普通个 \$ 記號，愛會記得頭前愛加扑一个反斜線 \。
