任何專業个網頁程式設計師，攏應該愛知影一位美國儂个大名 Brendan Eich /ˈaɪk/：二十年前，伊用十工个時間，替 Netscape 公司設計出 JavaScript 程式語言个初（粗）版，通好赴 Navigator 2.0 瀏覽器上市个時間。進前个網頁，主要攏是靜態个文圖，瀏覽器內建 JavaScript 了後，tsiah有才調予儂製作有動態互動功能个網頁。

微軟公司拄開始 khòo-khòo，無關心「三字W」（網際網路）个市場潛力，一旦智覺毋是勢，也仿做出一套仝功能个 JScript 佇 IE 瀏覽器內底，IE 閣佮Windows縛做伙，微軟利用伊作業系統个市場優勢，無幾年个期間，就扑敗 Netscape，佇瀏覽器个戰爭大刣贏。

Eich 怹一陣儂改變經營思想，建立 Mozilla 計畫，將瀏覽器相關个軟體源碼開放予社群，用自由自主、交流分享个軟體開發模式來對抗封閉自利、頂司管下司、鋤頭管畚箕个軟體產業。

聚合[tsū-ha̍p]全世界濟濟軟體社群个努力，所形成个「自由開放軟體」運動，會得著大成功，造福全人類，Mozilla 基金會所徛个地位佮影響不止重要。

Eich 因為伊个貢獻，受著業界个敬重佮關注，伊个職務也一直高升，到今年三月間，出任 Mozilla 公司个總經理。

結果，袂直啦！兩年前有儂搤空[iah-khang] Eich 佇 2008 年，私人捌寄付一千美金（寄付名單有十萬儂！），支持加州議會修法，欲明文規定一男一女个婚姻關係才是合法个。今伊欲做 Mozilla 个總經理，就引起公司內部佮外界極大个不滿佮抗議。

Eich 升做 CEO 到甲第九日就看破辭職，無夠十日。

即種代誌，若發生佇台灣，的確袂按呢生啦。
