我是 lynda.com 付費个會員，怹最近有出介紹 web font 最新發展个影片 ("Web Fonts First Look")，即類个技術，予使用者毋免先踮家治个電腦安裝某一種字型，嘛會凍佇看網頁个時，佇背景即時下載、顯示，嘛會凍予咱當做文字「複製/貼起去、tshiau-tshiāu」等等來使用，毋是用圖檔顯示奇巧 lè-táu 个特殊字體。

親像 http://lostworldsfairs.com/ 展示三張首頁个效果按呢，汝若用卡新个瀏覽器[liû-lâm-khì]點入去看，內底每一个字母攏thìng好選擇、反白、走揣[tsáu-tshuē]。

微軟个 IE 正式版到 tann (IE8) 一直攏干焦支援[tsi-uān]一種 web font 个格式，著是伊關門自製个 eot 。若別種瀏覽器，著有支援 ttf 抑是 otf 即寡普通个字型格式，毋過無法度用 MS 个 eot 來做 web font。

W3C 佇 2010 年制定出 web font 个新標準，woff。這 Chrome 佮 Firefox 攏會用--得矣。毋過，MS 愛等到 IE9 出來才會開始支援，著準 IE9 上市，嘛愛幾若年後，使用者才會漸漸昇級（愛先昇級作業系統到 win7 才會用得裝），到迄陣，網頁製作者採用 web font 才毋免遐麻煩，一種字型攏著攢[tshuân]幾若種格式。

論到 IE 對 CSS3 標準个支援，彼閣愈漚古。IE6/7/8 等一是完全袂用得，IE9 个進益[tsìn-iah] 嘛閣綴現時其他个瀏覽器袂著。

微軟為欲維護伊孤門獨市[koo-mn̂g to̍k-tshī]个利益，一直咧 thua-thuī 網路佮軟體个進步。即點咱凊彩看一份外文个資料攏感受會出，但是台灣个從業人員，著無一定知影即點；知，嘛無影想欲去改變。這佮台灣儂對殖民統治者个態度足sîng。
