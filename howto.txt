掠 Blogger.com 个 xml feed，轉做 _posts/*.html 檔，這毋是全部个資料
$ ruby blogger_to_jekyll.rb https://khoguan.blogspot.com/feeds/posts/default

家己下載 khoguan.blogpost.com 个 xml 備份，了後用 Jekyll import 來轉換：
https://import.jekyllrb.com/docs/blogger/

Jekyll-theme-chirpy 个安裝路徑：
$ cd /Users/khoguan/.gem/ruby/3.1.4/gems/jekyll-theme-chirpy-5.6.1
 
---
# 用 Jekyll themes 建置个網站，檔案攢好勢，設定檔（上重要个是 _config.yml）改好了後，
# 著愛佇網站根目錄執行：

$ bundle exec jekyll build

$ bundle exec htmlproofer _site --disable-external --check-html --allow_hash_href

$ bundle exec jekyll serve

$ bundel exec jekyll serve --livereload
# 欲緊，執行最後迄逝就好。
